## How to run

Please follow the below instructions to run this project in your machine:

1. Clone this repository
   ```sh
   git clone https://github.com/iamfahim07/Quiz-App-Backend.git
   ```
2. Install node_modules
   ```sh
   npm install
   ```
3. Run the app
   ```sh
   npm start
   ```
